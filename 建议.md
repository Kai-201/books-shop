## 1. BCrypt密码加密

问题：

当前密码明文存储。

改造：

注册：

```java
passwordEncoder.encode(password)
```

登录：

```java
passwordEncoder.matches(rawPassword, encodedPassword)
```

收益：

* 防止数据库泄露导致密码暴露
* 企业项目标准做法

## 库存超卖问题

场景：

某本书库存：

```text
stock = 1
```

两个用户同时下单。

---

### 方案一：乐观锁

```java
@Version
private Integer version;
```

特点：

* 性能高
* 适合读多写少

---

### 方案二：悲观锁

```sql
select * from book
where id=?
for update;
```

特点：

* 安全
* 并发性能较低

---

面试价值：

★★★★★
